Wolf-React

Wolf-React is a web application designed to assist players with the golf betting game "Wolf". This was a group assignment, so all server-side javascript and database code was written by myself, while the UI and overall design was made by my partners. The server uses MongoDB to store multiple sessions at the same time, while the client uses React. The application is limited in capability and scope, but still fully-functional for 4 players.
